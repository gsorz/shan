# 键值互取

  • 原理
var Color;
(function(Color) {
  Color[(Color['Red'] = 0)] = 'Red';
  Color[(Color['Green'] = 3)] = 'Green';
  Color[(Color['Blue'] = 4)] = 'Blue';
})(Color || (Color = {}));
let c = Color.Blue;
let d = Color[0];
console.log(c, d);

Color = {
  0: "Red"，
  3: "Green"，
  4: "Blue"，
  Blue: 4，
  Green: 3，
  Red: 0，
}

# Object

object表示非原始类型，也就是除number，string，boolean，symbol，null或undefined之外的类型。
使用object类型，就可以更好的表示像Object.create这样的 API。例如：

declare function create(o: object | null): void;

create({ prop: 0 }); // OK
create(null); // OK

create(42); // Error
create('string'); // Error
create(false); // Error
create(undefined); // Error

# 与 Object 区别

Object 类型的变量只是允许你给它赋任意值 - 但是却不能够在它上面调用任意的方法，即便它真的有这些方法

# never

# 永远不存在的值

// 返回never的函数必须存在无法达到的终点
// 抛异常
let error = () => {
  throw new Error('error');
};
// 返回never的函数必须存在无法达到的终点
// 死循环
let endless = () => {
  while (true) {}
};
// 推断的返回值类型为never
function fail() {
  return error('Something failed');
}
